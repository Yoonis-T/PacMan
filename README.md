# 🟡 PacMan Unity Game

A classic 2D PacMan-style arcade game built in Unity. Eat pellets, avoid ghosts, collect power-ups, and clear the maze to win!

---

## 🎮 Features

- Classic PacMan gameplay mechanics  
- Smooth player movement through maze corridors  
- Collectible pellets with scoring system  
- Power pellets that weaken ghosts temporarily  
- Enemy ghost AI with chase behavior  
- Lives system and game over screen  
- Multiple levels/maps support  
- Simple retro 2D arcade design  
- UI system with score and lives display  

---

## 🕹️ Controls

### Player

| Action | Key |
|---|---|
| Move Up | `W` or `↑` |
| Move Down | `S` or `↓` |
| Move Left | `A` or `←` |
| Move Right | `D` or `→` |

---

## 👻 Gameplay

- Move through the maze and eat all pellets  
- Avoid ghosts or lose a life  
- Collect power pellets to make ghosts vulnerable  
- Eat vulnerable ghosts for bonus points  
- Clear all pellets to complete the level  
- Lose all lives = Game Over  

---

## ⚡ Power-ups

| Power-up | Effect |
|---|---|
| 🔵 Power Pellet | Temporarily weakens ghosts |
| ⚡ Speed Boost | Increases PacMan movement speed |
| ⭐ Bonus Fruit | Gives extra score points |

---

## 🧠 Enemy AI

- Ghosts patrol and chase the player  
- Different ghost movement behaviors  
- Vulnerable state after power pellet pickup  
- Respawn system after ghosts are eaten  

---

## 🛠️ Built With

- Unity Engine (C#)  
- 2D Tilemap System  
- Custom PacMan & Ghost AI scripts  
- Unity UI system  

---

## 🚀 How to Run

1. Clone or download the project  
2. Open the folder in Unity Hub  
3. Open the main scene (`GameScene`)  
4. Press **Play** in the Unity Editor
